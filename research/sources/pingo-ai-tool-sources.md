# Pingo — AI Tool Source List

The watch list for Pingo's weekly "What's New in AI" research. Scope is deliberately narrow: **AI tools a small-business owner actually uses to market themselves.** If an update wouldn't change how a non-marketer owner works this week, it doesn't belong here.

This is the one file to keep current as the toolscape shifts. Add a source when a tool earns a place in the Pingo Stack; retire one when it stops mattering to owners.

**How the research skill uses this:** [`.claude/skills/pingo-trend-research.md`](../../.claude/skills/pingo-trend-research.md) sweeps every source below each week, keeps only verified shipped updates, and translates them into the weekly digest.

---

---

## Maintenance
- **Owner-value filter:** before adding a source, ask "would a busy owner with no marketing background care about this tool's updates?" If no, leave it out.
- **Primary-source rule:** aggregators are leads only. Every claim that reaches a Pingo post must be traced to and cited from the tool's own announcement.
- Review this list quarterly alongside the Pingo Stack.
