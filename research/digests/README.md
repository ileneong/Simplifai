# Intel Digests

Weekly "What's New in AI" digests produced by the [`pingo-trend-research`](../../.claude/skills/pingo-trend-research.md) skill.

- **Naming:** `DRAFT_intel-digest_[YYYY-MM-DD].md` (date = the Monday of the week).
- **Source of truth for what to watch:** [`../sources/pingo-ai-tool-sources.md`](../sources/pingo-ai-tool-sources.md).
- **Cadence:** weekly. Intended to run hands-off via a scheduled cloud routine (`/schedule`); can also be produced on demand by invoking the research skill.

Each digest gives you Top 3 owner-relevant updates (each sourced + dated + translated), a watch list, and one ready-to-build carousel brief. Pick an item, then build with [`pingo-instagram-post.md`](../../.claude/skills/pingo-instagram-post.md).
