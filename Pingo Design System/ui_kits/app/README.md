# Pingo App UI Kit

A high-fidelity component library and interactive prototype for the Pingo web app.

## Screens
1. **Onboarding / Welcome** — hero + CTA to get started
2. **Dashboard** — overview of active campaigns, quick actions
3. **Content Composer** — AI-powered post creation flow
4. **Campaign Detail** — metrics, post list, schedule view
5. **Settings** — profile, integrations, billing

## Components
- `Sidebar` — main navigation
- `TopBar` — page header with actions
- `StatCard` — metric display card
- `PostCard` — content post preview
- `CampaignRow` — list item for campaigns
- `ComposerPanel` — AI content generation panel
- `Button`, `Input`, `Badge`, `Toggle` — base UI atoms

## Fonts
- Poppins (headings, labels) — Google Fonts
- Plus Jakarta Sans (body, UI) — Google Fonts
