# Content Calendars

Monthly publishing plans per client.

## Naming
`CLIENTNAME_content-calendar_YYYY-MM.xlsx` or `.md`
Example: `green-roots_content-calendar_2026-05.xlsx`

## What Each Calendar Should Include
- Post date and time
- Platform
- Content type (caption, reel, carousel, story)
- Topic / theme
- Copy status (draft / approved / scheduled / published)
- Visual brief notes
