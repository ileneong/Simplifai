# Content

All content production lives here, organised by status.

- `drafts/` — Work in progress. Prefix files with `DRAFT_`
- `approved/` — Client-approved content ready to schedule. Prefix with `APPROVED_`
- `published/` — Archive of live/posted content

## File naming
`CLIENTNAME_PLATFORM_CONTENT-TYPE_YYYY-MM-DD.md`
Example: `green-roots_instagram_caption_2026-04-15.md`
